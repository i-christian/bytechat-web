# ByteChat Web Client

This repo provides the frontend for the ByteChat repo

## Requirements

- Node v20
- npm

## Usage

Clone
```shell
$ git clone https://github.com/i-christian/bytechat-web.git
$ cd bytechat-web
```

Then install the depencencies using npm

```shell
$ npm install
```

Finally, run the development server

```shell
$ npm run dev
```
